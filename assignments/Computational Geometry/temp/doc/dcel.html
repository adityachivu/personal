Doubly Connected Edge List

Objective:

To create a DCEL data structure API to represent Meshes and Subdivisions and perform relevant operations and queries. A new DCEL is initialized by creating an object of class Mesh. Mesh consists of the following classes.

  1. HalfEdge
  2. Vertex
  3. Face
  4. EdgeIterator

Design Principles

Here we take a look at a breakdown of our code and explain our approach.

  • createHalfEdge/ createVertex/ createFace

    These create the required pointers and push them into the Mesh structure. The Mesh maintains a list of all the Faces and Vertices and HalfEdges in a vector

  • createPolygonMesh

    This method creates a two-faced (pun intended) Mesh, an exterior and interior face. and updates required HalfEdge and Vertex pointers.

  • triangulate

    This method first makes a call to makeYmonotone and the triangulates each resulting face.
